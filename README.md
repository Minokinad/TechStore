# TechStore — Интернет-магазин электроники

## Описание проекта

**TechStore** — это современный веб-проект интернет-магазина, разработанный в рамках лабораторной работы. Проект демонстрирует навыки настройки рабочего окружения, проектирования интерфейсов в Figma и базовой верстки с использованием современных стандартов разработки.

## Используемые технологии

- **Frontend:** HTML5, CSS3 (Flexbox, Grid, CSS Filters)
- **Scripting:** JavaScript (ES6+)
- **Design:** Figma (использована концепция Soft Glow с радиальными градиентами)
- **Version Control:** Git & GitHub
- **Environment:** Node.js & npm

## Структура проекта

- `index.html` — главная страница магазина.
- `css/style.css` — стили проекта, включая адаптивную сетку и визуальные эффекты.
- `js/script.js` — базовая интерактивность (обработка кликов, консольные логи).
- `assets/` — статические ресурсы проекта.
- `images/` — изображения категорий товаров и графические элементы.
- `design/` — ссылка на макет Figma и документация по дизайну.

## Особенности реализации

1.  **Интеграция дизайна:** Реализован бесшовный радиальный фон (Glow Effect) на основе макета из Figma.
2.  **Адаптивность:** Сетка категорий товаров (Smartphones, Laptops, Audio и др.) автоматически подстраивается под размер экрана.
3.  **Контроль качества:** Код отформатирован с помощью _Prettier_, настроен файл `.gitignore` для исключения системного мусора из репозитория.

## Как запустить проект

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/Minokinad/TechStore
    ```
2.  Установите зависимости (если требуется):
    ```bash
    npm install
    ```
3.  Запустите проект с помощью расширения **Live Server** в VS Code.
